h1. Itsy Bitsy Gallery

*_version 0.7_* _(updated 23 November, 2011)_

h3. What does it do?

It displays all the image files within a folder as a gallery.

h3. Configurations?

_None_! Put the file in the folder, and see magic unfold!

*Alternatively*, if you want to configure some options, open up @index.php@, and edit these lines 19 through 25. :)

If you want to edit the CSS, it starts from line # 118 in @index.php@.

h3. Requirements?

Images. PHP.
