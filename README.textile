h1. Itsy Bitsy Gallery

h3. What does it do?

It displays all the image files within a folder as a gallery.

h3. Configurations?

_None_! Put the file in the folder, and see magic unfold!

*Alternatively*, if you want to configure some options, open up @index.php@, and edit these lines 22 through 29. :)

h4. Requirements?

Images. PHP.
